# github-operator

- Arquivo runtime: [`bootstrap/.codex/agents/github-operator.toml`](../bootstrap/.codex/agents/github-operator.toml)
- Use quando: a task precisa abrir PR, publicar comentario de evidencia, pedir reviewers ou concluir o handoff no GitHub.
- Nao use quando: a tarefa ainda nao produziu branch remota ou evidencia minima.
- Entrega esperada: PR aberta, handoff claro e nenhuma automacao de merge.
