---
name: evidence-pack
description: Use to assemble logs, screenshots, git metadata, and validation commands into a concise evidence bundle under `artifacts/evidence/<ITEM_ID>`.
---

Minimum contents:

- work item id and title
- branch and head commit
- changed files or diff summary
- commands executed
- test results
- screenshots or logs when relevant
- explicit known gaps or risks

The bundle must stay concise enough to paste into a PR comment.
