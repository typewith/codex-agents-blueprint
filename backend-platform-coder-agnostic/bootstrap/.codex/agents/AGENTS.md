# Runtime agents

Use estes agentes como especialidades executoras dentro do bootstrap `backend-platform-coder-agnostic`.

- `architecture-lead`: Architecture-focused agent for backend boundaries, tradeoffs, rollout posture, and decision quality.
- `contract-designer`: Contract-focused agent for APIs, messages, interfaces, and compatibility expectations.
- `data-structure-reviewer`: Review-focused agent for data structures, invariants, and state transition clarity.
- `integration-planner`: Planning-focused agent for sequencing, boundary cuts, and dependency risk.
- `risk-reviewer`: Review-focused agent for failure modes, degraded behavior, rollout, and rollback posture.
- `evidence-curator`: Evidence-focused agent for decision summaries, readiness notes, and PR artifacts.
- `github-operator`: GitHub-focused agent for draft PR publication, reviewer routing, and review thread hygiene.
- `reviewer`: Review-focused agent for bugs, missing validation, behavioral regressions, and unclear decisions.
