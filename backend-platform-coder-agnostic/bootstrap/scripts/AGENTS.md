# Scripts

Scripts operacionais para `backend-platform-coder-agnostic`.

- `scripts/backlog/`: work-item preparation, execution, review, and cleanup
- `scripts/github/`: draft PR publication and review follow-up
- `scripts/qa/`: evidence capture and validation notes
- `scripts/worktree/`: local environment linking for isolated worktrees
- `scripts/lib/`: shared helpers for CLI parsing, git, GitHub, templates, and work-item resolution
