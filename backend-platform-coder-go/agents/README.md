# Agents

Catalogo publico dos subagentes incluidos em `backend-platform-coder-go`.

- [`backend-architect`](./backend-architect.md): define boundaries, contratos e estrategia de rollout de servicos Go
- [`go-service-implementer`](./go-service-implementer.md): implementa handlers, servicos e fluxos operacionais em Go
- [`api-contract-designer`](./api-contract-designer.md): desenha contratos e compatibilidade para servicos Go
- [`storage-reviewer`](./storage-reviewer.md): revisa storage, schema e risco de persistencia
- [`concurrency-reviewer`](./concurrency-reviewer.md): revisa concorrencia, retries e seguranca de workers
- [`integration-validator`](./integration-validator.md): valida integracoes e confianca operacional do servico
- [`evidence-curator`](./evidence-curator.md): organiza evidence pack e notas de release
- [`github-operator`](./github-operator.md): publica e mantem o PR draft
