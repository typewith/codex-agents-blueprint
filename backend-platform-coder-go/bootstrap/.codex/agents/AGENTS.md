# Runtime agents

Use estes agentes como especialidades executoras dentro do bootstrap `backend-platform-coder-go`.

- `backend-architect`: Architecture-focused agent for Go services, boundaries, contracts, and rollout decisions.
- `go-service-implementer`: Implementation-focused agent for Go services, handlers, package boundaries, and operational code paths.
- `api-contract-designer`: Contract-focused agent for APIs, messages, and compatibility expectations in Go services.
- `storage-reviewer`: Review-focused agent for storage assumptions, schema changes, and persistence risk.
- `concurrency-reviewer`: Review-focused agent for goroutines, worker patterns, retries, and concurrency safety.
- `integration-validator`: Validation-focused agent for service checks, smoke tests, and integration confidence.
- `evidence-curator`: Evidence-focused agent for logs, schema notes, concurrency notes, and release artifacts.
- `github-operator`: GitHub-focused agent for draft PR publication, reviewer routing, and thread hygiene.
