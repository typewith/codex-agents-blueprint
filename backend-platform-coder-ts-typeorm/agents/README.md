# Agents

Catalogo publico dos subagentes incluidos em `backend-platform-coder-ts-typeorm`.

- [`backend-architect`](./backend-architect.md): define boundaries, contratos e estrategia de rollout do backend
- [`api-contract-designer`](./api-contract-designer.md): desenha contratos de API e compatibilidade
- [`typeorm-implementer`](./typeorm-implementer.md): implementa entidades, repositorios e migrations em TypeORM
- [`migration-reviewer`](./migration-reviewer.md): revisa risco de migration e reversibilidade
- [`integration-validator`](./integration-validator.md): valida integracoes, smoke tests e confianca operacional
- [`evidence-curator`](./evidence-curator.md): organiza evidence pack tecnico e notas de handoff
- [`github-operator`](./github-operator.md): publica e mantem o PR draft
- [`reviewer`](./reviewer.md): faz a revisao tecnica final da mudanca
