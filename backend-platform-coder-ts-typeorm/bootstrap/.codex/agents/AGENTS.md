# Runtime agents

Use estes agentes como especialidades executoras dentro do bootstrap `backend-platform-coder-ts-typeorm`.

- `backend-architect`: Architecture-focused agent for Node services, module boundaries, contracts, and rollout decisions.
- `api-contract-designer`: Contract-focused agent for request and response shapes, compatibility, and versioning discipline.
- `typeorm-implementer`: Implementation-focused agent for TypeORM entities, repositories, queries, and migrations.
- `migration-reviewer`: Review-focused agent for schema change risk, reversibility, and rollout caveats.
- `integration-validator`: Validation-focused agent for end-to-end checks, smoke tests, and integration confidence.
- `evidence-curator`: Evidence-focused agent for command logs, migration notes, contract summaries, and PR artifacts.
- `github-operator`: GitHub-focused agent for draft PR publication, reviewer routing, and review thread hygiene.
- `reviewer`: Review-focused agent for bug finding, missing tests, behavioral regressions, and technical risk.
