# Agents

Catalogo publico dos subagentes incluidos em `design-system-coder`.

- [`design-system-architect`](./design-system-architect.md): define o fluxo de prototipo, inventario de componentes e contratos de handoff
- [`mock-flow-builder`](./mock-flow-builder.md): monta mocks navegaveis e estados de tela em codigo
- [`component-implementer`](./component-implementer.md): implementa componentes e encaixa tokens no mock
- [`visual-reviewer`](./visual-reviewer.md): revisa hierarquia visual, composicao e paridade com o brief
- [`accessibility-reviewer`](./accessibility-reviewer.md): faz a revisao de acessibilidade do mock e dos componentes
- [`evidence-curator`](./evidence-curator.md): organiza screenshots, notas de paridade e evidencias do work item
- [`github-operator`](./github-operator.md): abre o PR e mantem a conversa de review organizada
