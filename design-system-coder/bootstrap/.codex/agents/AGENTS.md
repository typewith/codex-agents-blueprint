# Runtime agents

Use estes agentes como especialidades executoras dentro do bootstrap `design-system-coder`.

- `design-system-architect`: Architecture-focused agent for prototype contracts, token strategy, mock structure, and handoff boundaries.
- `mock-flow-builder`: Execution-focused agent for building code-first mocks, stateful screens, and interaction shells.
- `component-implementer`: Implementation-focused agent for reusable components, tokens, and low-risk UI assembly.
- `visual-reviewer`: Review-focused agent for hierarchy, composition, spacing, and prototype parity checks.
- `accessibility-reviewer`: Accessibility-focused agent for semantic structure, keyboard flow, contrast, and state clarity.
- `evidence-curator`: Evidence-focused agent for screenshots, parity notes, change summaries, and review artifacts.
- `github-operator`: GitHub-focused agent for PR publication, reviewer routing, and response hygiene.
