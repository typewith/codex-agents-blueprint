# Agents

Catalogo publico dos subagentes incluidos em `mobile-app-coder`.

- [`mobile-architect`](./mobile-architect.md): define arquitetura mobile, navegação e fronteiras de release
- [`react-native-implementer`](./react-native-implementer.md): implementa telas, componentes e fluxos em React Native/Expo
- [`android-studio-specialist`](./android-studio-specialist.md): cobre Android Studio, emulator e caveats nativos de Android
- [`xcode-specialist`](./xcode-specialist.md): cobre Xcode, simulator e caveats nativos de iOS
- [`api-integration-validator`](./api-integration-validator.md): valida consumo de API e integracoes no app
- [`visual-evidence-reviewer`](./visual-evidence-reviewer.md): revisa paridade visual e ajuste das telas
- [`release-curator`](./release-curator.md): organiza evidencia e notas de release para mobile
- [`github-operator`](./github-operator.md): publica e mantem o PR draft
