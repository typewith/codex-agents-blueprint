# Runtime agents

Use estes agentes como especialidades executoras dentro do bootstrap `mobile-app-coder`.

- `mobile-architect`: Architecture-focused agent for mobile app surface, navigation structure, state coverage, and release boundaries.
- `react-native-implementer`: Implementation-focused agent for React Native or Expo screens, components, and stateful flows.
- `android-studio-specialist`: Tooling-focused agent for Android Studio, emulator assumptions, Android-specific checks, and native build caveats.
- `xcode-specialist`: Tooling-focused agent for Xcode, simulator assumptions, signing, provisioning, and iOS-specific caveats.
- `api-integration-validator`: Validation-focused agent for API sync, contract consumption, and mobile integration confidence.
- `visual-evidence-reviewer`: Review-focused agent for screen parity, hierarchy, device-fit issues, and release-facing visual notes.
- `release-curator`: Evidence-focused agent for release notes, mobile validation artifacts, and PR handoff clarity.
- `github-operator`: GitHub-focused agent for draft PR publication, reviewer routing, and review thread hygiene.
