# Scripts

Scripts operacionais para `mobile-app-coder`.

- `scripts/backlog/`: generic work-item helpers retained for backlog-first and task-first flows
- `scripts/mobile/`: screen preparation, screen review, and native-tooling checks
- `scripts/github/`: draft PR publication and review follow-up
- `scripts/qa/`: mobile evidence capture and validation notes
- `scripts/worktree/`: local environment linking for isolated worktrees
- `scripts/lib/`: shared helpers for CLI parsing, git, GitHub, templates, and work-item resolution
